# 🛰️ ARO Sentinel: Jade Rewards Calculator

A professional, lightweight tool designed to help the **ARO Network** community estimate their potential earnings during **Testnet Sprint 2 (S2)**.

## 📖 Overview
As ARO Network scales its decentralized edge cloud, rewarding participants through the **Jade points** system is central to its growth. This calculator provides node operators with a clear way to quantify their contributions and plan their hardware deployments.

## ✨ Features
*   **Node Type Presets**: Pre-configured rates for ARO Mobile, Desktop, Pod, and Server nodes.
*   **Dynamic Uptime Scaling**: Calculate earnings based on real-world active hours.
*   **Referral Bonus Integration**: Includes the **15% direct referral bonus** to show total potential growth.
*   **Mobile-First Design**: Optimized with Tailwind CSS for seamless use on any device.

## 🧮 How It Works
The calculator uses the latest Sprint 2 quantification rules:
1. **Base Rate**: Determined by the specific node type (e.g., ARO Desktop vs. ARO Mobile).
2. **Uptime**: Multiplies the base rate by your projected active hours per day.
3. **Multipliers**: Applies the referral commission (15%) to your total base earnings.

## 🚀 Getting Started
### Deployment
This tool is a standalone HTML/JavaScript application. You can host it for free on **GitHub Pages**:
1. Fork this repository.
2. Go to **Settings > Pages**.
3. Select the `main` branch as the source and click **Save**.

### Local Use
Simply clone the repo and open `index.html` in any modern web browser.

## 🛠️ Built With
*   **Tailwind CSS**: For a modern, professional UI.
*   **JavaScript (ES6)**: For real-time, responsive calculation logic.
*   **Font Awesome**: For intuitive iconography.

## 🛡️ Disclaimer
This tool provides **estimates only**. Actual Jade distribution is dynamic and depends on the total number of active participants in the global 100M daily Jade pool.
